exemplo de CRUD com JSP e Servlet

https://danielniko.wordpress.com/2012/04/17/simple-crud-using-jsp-servlet-and-mysql/


- criar dynamic web project

- Converter para Maven project (botão direito > Configure > Convert to Maven Project)

- Adicionar dependências de BD e JSTL (jstl 1.2 maven / standard 1.1.2 maven)

- Configurar o servidor (botão direito no projeto > Properties > Project Facets > Runtimes)

- Copiar classes do projeto DAO para o projeto CRUD

- Testar conexão

- Implementar JSP's (Tela inicial, Listagem e Inclusão/Edição)

- Implementar servlet
